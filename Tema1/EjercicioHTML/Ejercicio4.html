listas de definiciones

  • URL

    Una URL es la dirección que escribes en el navegador para visitar una página web o acceder a un recurso en línea.

  • HTTP

    Es el protocolo de comunicación que permite la transferencia de datos en la web. Se utiliza para que el navegador solicite y reciba páginas web desde un servidor.

  • TCP/IP

    Es el conjunto de protocolos que permite la comunicación entre computadoras en Internet. TCP se encarga de asegurar que los datos lleguen completos y en orden. IP se encarga de direccionar y enviar los datos al destino correcto.

  • Internet

    Internet es una red mundial de redes de ordenadores que permite la comunicación e intercambio de información entre millones de dispositivos en todo el planeta mediante un conjunto de normas llamadas protocolos de comunicación, siendo el más importante TCP/IP.

  • W3C

    Es la organización internacional que desarrolla los estándares y buenas prácticas para la web (como HTML, CSS, etc.).

  • HTML

    Es el lenguaje de marcado estándar que se usa para crear páginas web. Define la estructura del contenido usando etiquetas etc.
